Student Desktop Application

Made with Unity

Important Files:

StudentApp(Standalone).exe - This is the single standalone .exe file that will allow you to run the game.
Local database have to be running concurrently for the game to work

/Assets/LoginController.cs - This is the script that is running the login page screen
/Assets/MainGameController.cs - This is the script that is running the main game page.
/Assets/QuizController.cs - This is the script that is running the quiz page
/Assets/ChallengeController.cs - This is the script that is running the challenge page to choose who to challenge.
